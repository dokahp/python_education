Description

You're working on a college newspaper called HyperNews. You are comfortable in the digital world, so you came up with the idea of publishing the news online and decided to create a website for this.


Creating the HyperNews website will take some time, so for now you’ll create a “Coming soon” page.

Objectives

Create the first handler for your server.

If you start the project on your computer with the python manage.py runserver command, your server should respond with the text "Coming soon" at the localhost:8000. address.
